Construction
  • Container — 82×82 rounded square, 24px corner radius, 9px stroke. Open form, never filled, so it reads as a window/frame.
  • Glyph — one upright rounded bar — the i of indie. Pill caps, set left-of-centre like a spine.
  • Clearspace — keep at least the corner-radius (≈¼ icon) clear on every side.
  • Minimum size — 20px. Below that, use the filled tile variant.
